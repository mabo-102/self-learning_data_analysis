## DX

- 何を目指すのか目的を明確にする
- 目的を達成する通過点として目標を置く
- 目的達成のために限られた資源（ヒト・モノ・カネ）をどう有効活用するのかの指針として戦略を考える
- 目的達成の阻害要因として問題がある
- 問題を解決するためにやるべきことが課題
